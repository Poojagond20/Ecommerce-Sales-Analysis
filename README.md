Ecommerce Sales Analysis Dashboard

📊 Project Overview

This project presents an **interactive Ecommerce Sales Dashboard built in Microsoft Excel**.
The dashboard analyzes sales performance and highlights key business insights using visual charts and KPI indicators.

The objective of this project is to convert raw ecommerce data into meaningful insights that support business decision-making.


📁 Project Files

* Ecommerce Sales Analysis.xlsx – Main dashboard file
* dashboard.png – Dashboard preview image
* README.md – Project documentation

📈 Dashboard Features

* Sales KPI metrics
* Year-over-Year comparison
* Top 5 product/category analysis
* Geographic sales distribution
* Sales trend analysis
* Data visualization using charts


 Tools & Skills Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization
* Dashboard Design


  Key Insights

* Identifies top-performing products
* Tracks yearly revenue growth
* Visualizes geographic sales patterns
* Highlights business performance metrics
